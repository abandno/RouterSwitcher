# 1. 需求 V2 优化

## 1.1. IP专用输入框

IP、网关、DNS，输入框样式改成专门输入IP设计的

1. "."分割成四个子输入框
2. 限定数字范围 0~255
3. 输入满3位数字，光标自动跳到后面一个子输入框里
4. 光标聚焦子输入框时，按Tab快捷也会调到后面的子输入框里

![image-20251203013530046](image/需求%20V2/image-20251203013530046.png)

## 1.2. 托盘图标根据当前IP模式自动切换

1. 自适应IP时，紫色的图标
2. 动态IP时，红色的图标
3. 静态IP时，蓝色的图标

## 1.3. 配置页表单

### 1.3.1. 提交前校验

1. 自适应 时，SSID 和 静态IP配置区域 必填
2. 静态IP 时，静态IP配置区域 必填


### 1.3.2. 配置页表单项联动 （挂起）

1. 当 IP模式 为 自适应 时，SSID 和 静态IP配置，应可编辑
2. 当 IP模式 为 动态IP时， SSID 和 静态IP配置，应变成禁用状态的样式，不可编辑
3. 当 IP模式 为 静态IP时，SSID 应为禁用状态样式，不可编辑，静态IP
